# Integrations

## Identity providers

SentinelAI supports SSO through standard identity providers via SAML and OIDC, and supports SCIM-based user provisioning for supported providers.

## Networking

Private networking and IP allowlisting are available for eligible enterprise deployments.

## Security tooling

SentinelAI supports forwarding security events to common SIEM and logging destinations.

## Custom integrations

Integrations beyond supported connectors require scoping before a timeline can be committed.

## Approved wording

We support SSO via SAML and OIDC, SCIM provisioning for supported providers, SIEM event forwarding, and private networking or IP allowlisting for eligible deployments. Custom integrations require scoping.

## Forbidden wording

Do not commit to an unlisted integration or connector without scoping and Product approval.
